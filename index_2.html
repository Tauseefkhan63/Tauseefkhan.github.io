Project Details

• Tool Usage: Employed Python libraries (NumPy, Pandas, Seaborn, scikit-learn) for system development, enabling data handling, visualization, and model creation.

• Model Design: Developed a logistic regression-based diabetes prediction model, using eight key variables: pregnancies, glucose levels, blood pressure, skin thickness, insulin levels, BMI, diabetes pedigree, and age.

• Dependent Variable: Employed the 'outcome' variable to indicate diabetes presence or absence, serving as the model's dependent factor.

• Model Accuracy: Achieved an 81% accuracy rate with the logistic regression model, demonstrating its proficiency in predicting diabetes based on the specified variables.

• Web Integration: Integrated the model into a user-friendly Django-based web application, simplifying data input and offering clear result visualization, enhancing accessibility for diabetes prediction.
